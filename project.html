Why are remote repositories necessary?

Repositories is a place, room or container where something is deposited or stored.

Here is a a simplified breakdown of why Github repositories are important:

  • Central Hub: They hold all project files, documentation and version history
  • Collaboration: Multiple people can work on the project at the same time
  • Track changes: You can see who made what changes and when.
  • Pull Request: Propose updates and get them reviewed before merging.
  • Branches: Work on different features and versions without messing up the main code.
